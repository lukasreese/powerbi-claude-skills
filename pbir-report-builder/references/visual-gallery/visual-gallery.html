IBCS Custom Visuals (4 Templates)

These are not standard Power BI visuals. Each IBCS template is built from native charts with custom DAX measures and formatting. The skill generates all helper measures + PBIR JSON automatically from 3 inputs: actual measure, comparison measure, category/time column.
Actual (AC) — dark blue
Comparison (PY) — gray
Positive variance — green
Negative variance — red
1 — Column Variance Chart
lineClusteredColumnComboChart
Time-series comparison — AC columns, PY line, variance bars below baseline. Best for showing monthly/quarterly performance trends.
2 — Bar Variance Chart
barChart + NativeVisualCalculation
Ranked category comparison — horizontal AC bars with PY reference markers. Best for comparing many categories (products, regions, customers).
3 — Variance Table (Simple)
pivotTable + SVG measures
Numbers + SVG variance bars in the last column. Clean information density. Best for detailed breakdowns where exact values matter.
4 — Variance Table (Full)
pivotTable + SVG measures (full)
Full SVG table — AC + PY as proportional bars + variance column. Maximum information density. Best for executive summaries and management reporting.
